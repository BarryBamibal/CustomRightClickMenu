Custom Right-Click Menu

Welcome to Custom Right-Click Menu. This extension lets you customize a section in your right-click menu completely.
Some things you can do are creating a link to a website, executing a custom piece of JavaScript code or making a subsection with more things in it.
I recommend you first check out the entire options page before you start changing the actual items. This is mainly because there are a lot of things to make it easier for you. There are a few things you should know that are noted below.

I really recommend visiting the settings part when editing a script if you're planning on writing some javascript code. Some very important info is on there

Some scripts other people have created can be found on this page this page. If you feel you've created a script that a lot of people will enjoy, you can also share it so other people can get it too.
Because i don't have a site right now myself, and don't really know any good site to store these, you can share them on this page. If you've got suggestions for a site to use, please email them to me.



Options

Turn on the shortcut to the options page in your right-click menu

Open in current tab (if unchecked clicking a link will open it in a new tab)



Things you should know: This is what the colors and the see through items mean:

Red: Script
Blue: Link
Green: Menu
Yellow: Divider
See-through: A script that wont be shown in the right-click menu (one that does not run on click).


Don't edit scripts that you have not written, this will probably break them.

If the right-click menu is not updating after you changed something, reload the extension in the "extensions" page.

Click on this button to show a piece of code that allows you to search a search engine's website.





Note: If you want this URL for the "Add search engine" button, replace the %s with customrightclickmenu and keep the www.

You can replace "google" in the prompt ofcourse to what you want, and most importantly:
You can replace "https://google.com/search?q=" with any other search engine.
To find the url for this, go to settings, go to manage search engines and find the URL in the 3rd column.
Copy this, and remove www. from it and remove %s from it.
If the URL's format differs a lot from the google URL, take the domain and check if this works.
http://www.domain.com/search?q=
After the q= type any word, and see if it searches the site for that word.
If this worked, remove www. and place it between the parentheses ("")
Now try and see if this works, if it does, you just learned how to make a custom search.
I recommend making a menu with a lot of search engines in it to search easily.

Importing and Exporting

Exporting
Just copy the output into an import box.


Note: Say you are exporting rows 1 to 4 and number one is a menu with the next 5 being Child rows,
keep in mind that the person who imports it will also have the menu with 5 Child rows, affecting their other rows.
Also, say you select 1,2 and 4, the person who imports it will have row 3 as an empty row.






Importing
Copy export text into this box and hit the buton.


The imported rows will be added as new rows in the last place.
If you have 1-5 set, and import 3 rows, it will place them on place 6-8.
